# Vibes in the City

A mobile app prototype for discovering nightlife venues, events, and experiences in Tbilisi.

## Features

- 🗺️ Interactive map with venue pins
- 🔍 Search venues and events
- 📅 Events calendar — Tonight's Vibes
- ❤️ Save favourite places
- 👤 User profile
- 💼 Business Cabinet — manage venues, events, bookings, reviews, messages
- 🤖 AI-powered promo content generation (Claude API)
- 🎯 Table booking system

## Tech Stack

- Single-file HTML app
- Leaflet.js for maps
- Anthropic Claude API for AI features
- No build tools required

## Usage

Open `index.html` in any modern browser.

For AI features (Improve Description, Generate Promo Content), the app uses the Anthropic API — make sure you are running it through a proxy that provides the API key.
